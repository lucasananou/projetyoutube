# Site Vitrine Artisan Couvreur

Site vitrine professionnel orienté SEO local et génération de leads pour un artisan couvreur.

## Stack Technique

- **Next.js 15** (App Router)
- **React 19**
- **TypeScript**
- **Tailwind CSS**

## Getting Started

### Installation

```bash
npm install
```

### Développement

```bash
npm run dev
```

Ouvrir [http://localhost:3000](http://localhost:3000)

### Build Production

```bash
npm run build
npm start
```

## Configuration

Créer un fichier `.env.local` :

```env
NEXT_PUBLIC_SITE_URL=https://votre-domaine.fr
```

## Documentation

Consulter [ARCHITECTURE.md](./ARCHITECTURE.md) pour la documentation complète de l'architecture.

## Structure du Projet

- `src/app/` - Pages et routes (App Router)
- `src/components/` - Composants React réutilisables
- `src/config/` - Configuration des services, villes, blog
- `src/lib/` - Utilitaires et helpers
- `src/types/` - Types TypeScript

## Features

✅ SEO optimisé (metadata, OpenGraph, JSON-LD)  
✅ Pages dynamiques par service  
✅ SEO local par ville  
✅ Blog avec articles
